# Assignment-5

This repository contains Python programs created as part of Assignment 5 for Module 6: Data Structures and Strings in Python.

# Task 1: Create a Dictionary of Student Marks

This program:

Creates a dictionary where student names are keys and their marks are values

Takes a student name as input from the user

Checks whether the entered name exists in the dictionary

Displays the corresponding marks if found

Displays an appropriate message if the student is not found

# Task 2: Demonstrate List Slicing

This program:

Creates a list of numbers from 1 to 10

Extracts the first five elements using list slicing

Reverses the extracted elements

Displays both the extracted list and the reversed list
